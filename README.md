- 로그인
  - 클럽 목록
    - 클럽 상세
      - 클럽 정보
        - 운동시간
        - 회원 수
          - 회원목록 볼수 있는 페이지 이동 가능
        - 체육관 장소(text)
        - 기타(textArea)
        - 클럽 가입 여부
          - 가입 전: 가입버튼 활성화, 운동 참여 버튼 비활성화
          - 가입 후: 운동 참여 버튼 활성화
      - 운동 목록(오늘 운동 오니?)
        - 클럽 가입 여부
          - 가입 전: 가입버튼 활성화, 운동 참여 버튼 비활성화, 운동 상세 화면 비활성화
          - 가입 후: 운동 참여 버튼 활성화
        - 가입 진행
          - 가입 버튼 클릭 시 가입 진행 화면 표시
          - 가입 정보 입력
            - 이름
            - 이메일: 카톡에서 받은 정보 기본으로 보여줌
            - 사진: 카톡에서 받은 thumbnail 사진, 본인 얼굴이 아니면 본인 얼굴로 변경 추천
            - 휴대폰 번호
            - 성별
            - 생년월일
            - 구대회 신청 가능 급수
            - 전국대회 신청 가능 급수
            - 주소
          - 가입 진행 화면에서 가입 버튼 클릭 시 가입 완료
  - 관리자 페이지
    - 회원 목록
      - 가입 수락, 탈퇴 기능
    - 운동 일정 등록
      - 운동 목록에 나오게 됨
