# 회비 의무 시작 규칙 (가입 시기 반영)

## 목적

회원이 중간에 가입한 경우, **가입 이전 달 회비는 의무가 없음**을 시스템 전반에 반영하기 위한 규칙 정리.

---

## 1. 기본 원칙

- **기본**: 가입한 달부터 회비 의무 (승인 시 `feeObligationStartAt` = 승인 일시 → 해당 월부터 의무).
- **관리자 조정**: 상황에 따라 “이번 달부터” 또는 “다음 달부터” 납부하도록 **회비 입금 시작일**을 회원 상세에서 설정.
  - 이번 달부터: 해당 월 1일 등으로 설정.
  - 다음 달부터: 다음 달 1일로 설정.

---

## 2. 연도 경계

- **가입 연도가 당해 연도보다 이전** → 당해 연도는 **1월~12월 전 구간** 회비 의무.
- **당해 연도에 가입** → 당해 연도는 **첫 의무월~12월**만 회비 의무.
- **당해 연도 이후 가입** → 당해 연도에는 회비 의무 없음.

---

## 3. 부부 회원

- 부부 행(한 건 입금에 두 명)의 경우, **둘 중 늦은 의무 시작월**을 기준으로 함.
- 즉, 두 명 모두 회비 의무가 있는 달만 부부 행으로 집계/제안.

---

## 4. 관리자 설정

- 회원별 **회비 입금 시작일**(`feeObligationStartAt`)은 **관리자가 설정**할 수 있음.
- 설정 위치: **클럽 멤버 관리 → 회원 상세** (가입일 표시 + 회비 입금 시작일 설정).
- 회비 대시보드/처리 화면에서는 **현재 값 표시**만 하고, 수정 시 "회원 정보에서 수정" 링크로 이동.

---

## 5. 데이터

- `ClubMember.feeObligationStartAt` (DateTime?, nullable): 회비 의무가 시작되는 시점 (해당 연·월의 1일 등으로 설정해도 됨).
- 미설정 시: 당해 연도 1월부터 의무로 간주 (하위 호환).
- **승인 시**: status를 APPROVED로 변경할 때, `feeObligationStartAt`이 없으면 **그 시점(now())**으로 설정 (이미 있으면 유지).
